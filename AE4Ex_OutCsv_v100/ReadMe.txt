■ 概要
本ソフトを使用すると、アクションエディター4で作成されたステージファイルの変数名の一覧をCSVファイルに出力できます。
出力されたCSVファイルを表計算ソフトなどに取り込むことで、アクションエディター4とは別の画面で変数名を確認することができ、変数の管理を従来より効率的に行うことが期待できます。


■ インストール
ダウンロードしたファイルを解凍してください。


■ アンインストール
ダウンロードしたファイルを削除してください。


■ 使用方法
1. アクションエディター4で出力したステージファイル(.stg4_1020)を本ソフト(AE4Ex_OutCsv.exe)にドラッグ&ドロップすると、プログラムが起動します。
複数のステージファイルを選択してドラッグ&ドロップすると、複数のファイルに対して処理が実行されます。
<コマンドプロンプト上で行う場合>
本ソフトのパスの入力に続けて、入力ファイルのパスを入力してください。
例えば、カレントディレクトリが本ソフトと同じ位置にあるとき、相対パスで入力するなら以下のようになります。
AE4Ex_OutCsv.exe sample\sample_1.stg4_1020 sample\sample_2.stg4_1020
2. 出力ファイルと同名のファイルがある場合、上書きするか尋ねられます(画面表示：「[Output] Overwrite ?」)。
「Y」を入力し、エンターキーを押すと、上書きされます。
「N」を入力した場合は、上書きされません。
3. 問題が起きなければ、ステージ変数名の一覧が記述されたCSVファイルが作成されます(画面表示：「[Output] Finish」)。


■ 入力ファイル
ソフト：アクションエディター4
ソフトバージョン：10.20
ファイル：ステージファイル
拡張子：.stg4_1020


■ 出力ファイル
作成場所：(本ソフトがあるフォルダ)\out\
拡張子：.csv
ファイル名(拡張子込)：(入力ファイル名(拡張子込)).csv
ファイルの中身の形式：(変数番号),(変数名)


■ フォルダ構成
・AE4Ex_OutCsv_v*** (***はバージョン(「.」無し))
　・out (出力ファイルの格納フォルダです。プログラム実行時に作成されます。)
　・sample (動作確認用のファイルの格納フォルダです。)
　・AE4Ex_OutCsv.exe (本ソフトです。)
　・ReadMe.txt (このファイルです。)


■ 注意事項
1. 本ソフトの使用により何らかのトラブルが発生しても作者は一切の責任を負いかねます。
2. 上書きするか尋ねられた場合、「y」「Yes」「yes」を入力した場合は「Y」と同様になり、他の入力の場合は「N」と同様になります。
3. アクションエディター4(Ver.10.20)で作成されたステージファイルのみ対応いたします。
他のバージョンなどにより作成されたファイルは対応いたしません(未定義動作となります)。
4. 本ソフトや入力ファイルのパスにマルチバイト文字列(漢字など)を含む場合、失敗する場合がございます。
ファイルパスにマルチバイト文字列が含まれないようにしてください(相対パスで指定、ファイルを移動、リネーム、など)。
5. 入力ファイル名が長すぎる場合、正常に処理が実行されない場合がございます。
6. 入力ファイルの数が多すぎる場合、正常に処理が実行されない場合がございます。
7. ステージファイル以外のファイルを読み込んだ場合、そのファイルに対する処理は実行されず、CSVファイルは出力されません(画面表示：「[Read] This file is invalid.」)。
